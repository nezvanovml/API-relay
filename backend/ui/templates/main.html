Add a feature to your IoT device to control comfortably wherever it is, you only need working internet connection.

Get started with example for ESP32 device using Arduino IDE.

Watch example

Home Assistant-ready

You can install component for Home Assistant from GitHub and control your devices with smartphone or web application.

Absolutely free

This service are free, you don't need to pay for use it.

OTA firmware update

You can compile your code to binary, upload it to relay and device will update on next startup.

No data collecting

Relay does not save any info about your device (state or system info) on DB, it only transfers it from your server to device or to reverse.
